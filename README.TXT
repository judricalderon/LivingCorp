# 🏡 Inmobiliaria Web App



## Descripción

Bienvenido a la Inmobiliaria Web App, una solución completa para gestionar propiedades inmobiliarias. Esta aplicación permite a los usuarios buscar, ver y gestionar propiedades de manera eficiente. Está construida utilizando Java y Spring Boot para el backend, JSF para el frontend, y MySQL para la persistencia de datos.

## 🚀 Características

- **Gestión de Propiedades**: Crear, editar, eliminar y listar propiedades.
- **Búsqueda Avanzada**: Buscar propiedades según diferentes criterios como ubicación, precio, tipo, etc.
- **Autenticación y Autorización**: Gestión de usuarios con roles y permisos.
- **Interfaz Amigable**: Diseño intuitivo y responsivo usando JSF.
- **Consumo de Recursos HTTP**: Integración con servicios externos para obtener datos adicionales.
- **Notificaciones por Correo Electrónico**: Enviar notificaciones y respuestas por correo electrónico.

## 🛠️ Tecnologías Utilizadas

- **Java**
- **Spring Boot**
- **Jakarta EE (JSF, CDI)**
- **MySQL**
- **Hibernate**
- **Jakarta Mail**
- **Jakarta RESTful Web Services (JAX-RS)**
- **PrimeFaces**
- **Maven**

## 📦 Instalación y Configuración

### Prerrequisitos

- Java 11 o superior
- Maven
- MySQL
- Servidor de aplicaciones compatible con Jakarta EE (WildFly, Payara, etc.)
